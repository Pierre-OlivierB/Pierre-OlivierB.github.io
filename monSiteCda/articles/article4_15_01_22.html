Utilisation de GitHub en MULTI

Vérifier si on est à jour

Récupérez le travail fait sur le dossier commun

1- GitBashHere dans le dossier partagé et lié avec GitHub

2- $git pull
\\ copie le dossier sur GitHub

Modification sur GitHub

On peut modifier les fichier directement à partir de GitHub en cliquant sur le stylot à droite du fichier sélectionné.

Créez une branche

Sur son pc:

1- $git branch nomDeLaBranche
\\cela créé une branche nommée : nomDeLaBranche

2- $git branch
\\cela dévoile les noms des branches existantes, *nomEnVert c'est la position actuelle

3- $git checkout nomDeLaBranche
\\on se déplace sur la branche. Le nom de la branche apparaît maintenant à la place de (main) et est devenu (nomDeLaBranche).

Sur GitHub:

$git branch -a
\\afficher liste des branches présentes sur GitHub

Push une branche

Sur pc:

4- $git push –set-upstream origin nomDeLaBranche
\\double tiret devant "set".
!!!!Attention la branche doit être créer en local pour qu'elle puisse être push.!!!!

Sur GitHub

5- Apparition de la branche en question. On a désormais un milieu de travail indépendant du main.

Supprimer une branche en local

Normalement on fusionne donc cela n'est pas très utile!

$git branch -d nomDeLaBranche
\\!!!Attention : il ne faut pas être dessus pour la supprimer

Supprimer une branche distante

$git push origin --delete nomDeLaBranche
\\double tiret devant delete
