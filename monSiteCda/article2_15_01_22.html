Utilisation de GitHub

GitHub la plateforme

C'est quoi?

GitHub, c'est un site de partage de code où l'ont peut héberger son code, proposer des versions de code à d'autres utilisateurs et accepter des propositions pour son propre code.

Pour la suite allez vous inscrire sur -->GITHUB<--!

Pas à pas pour utiliser github:

Avant tout, quelques petites choses à savoir:

Repositories = projets.

Overview = apperçu du travail partagé.

Il y a un agenda sur l'année avec les mis à disposition sur github faites durant l'année avec des variances de couleur : plus c'est vert foncé plus il y a eu d'activité dans cette journée!

Création:

Allez sur git hub:

Dans la page Repositories, cliquer sur New (bouton vert à droite sur la page),

Owner:

C'est la personne qui l'a fait, si on est dans une entreprise il faudra sélectionner l'entreprise, si non c'est sur son compte personnel.

Repository name:

Nom du dossier du travail!

Public ou Private?

Le principe de github est de partager.

Git hub est gratuit quand on laisse Public et devient payant en Privé. Cela pour pousser au partage et avoir un modèle économique.

Initialize this repository with :

Add a README file:

On est dans du partage il faut donc expliquer au maximum son travail. Comment assembler un objet sans notice? Pensez aux autres. C'est la finalité de github.

Add .gitignore:

On ne veut pas forcément partager tous les fichiers (ex: les musiques qui ne sont pas libre de droit), il nous faut donc une sorte de blacklist pour empécher le partage de ces derniers même s'ils sont dans le dossier de travail.

Suppression d'un repository:

On va sur son projet puis:

Settings:

On scroll jusqu'en bas jusqu'à rencontrer un cadre rouge nommé : Danger Zone.

Delete this repository:

Une alrte s'affiche il faudra écrire une phrase pour confirmer la suppression!
